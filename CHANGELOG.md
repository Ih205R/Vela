# Changelog

All notable changes to Vela will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-18

### Added
- Initial release of Vela programming language
- Complete compiler with lexer, parser, and code generators
- CLI with `init`, `build`, and `dev` commands
- Development server with file watching and hot reload
- HTML5 generator with semantic markup
- Modern CSS generator with responsive design
- JavaScript generator with smooth scrolling and animations
- Standard library with components:
  - Hero section
  - Section container
  - Card component
  - Button component
  - Footer component
- Theme system with customizable colors
- Project scaffolding with `vela init`
- Configuration file support (`vela.config.json`)
- Comprehensive documentation and guides
- Example projects

### Language Features
- Site declaration
- Theme configuration
- Page routing
- Component composition
- String literals
- Comments (line and block)
- Property syntax
- Link navigation with `->` operator

### Developer Experience
- Helpful error messages with line numbers
- Colored CLI output
- File watching for development
- Local HTTP server
- Zero configuration defaults

### Documentation
- Complete README with examples
- Language guide (GUIDE.md)
- Contributing guidelines (CONTRIBUTING.md)
- MIT License

### Technical
- TypeScript implementation
- Node.js runtime
- Commander.js for CLI
- Chokidar for file watching
- Modular architecture
- Extensible design
